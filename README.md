# windowgrid

Виртуализация таблиц: в DOM держим только видимое окно строк и колонок, остальное не рендерим. Ядро на чистом TS без зависимостей, React-обёртка сверху по желанию.

Начал, когда в очередной раз уткнулся в тормоза большой таблицы и захотелось маленькое ядро, которое понимаю целиком, а не гигантский грид с сотней опций.

## Использование

```tsx
import { useWindowgrid } from "windowgrid/react";

const { viewport, scrollRef, measureRow, totalHeight, totalWidth } = useWindowgrid({
  rowCount: 1_000_000,
  columnCount: 40,
  estimatedRowHeight: 34,
  pinnedLeft: [0, 1],
});
```

Дальше вешаешь `scrollRef` на скролл-контейнер и раскладываешь `viewport.rows.items` / `viewport.columns.items` абсолютным позиционированием. Высоты строк меряются после рендера через `measureRow`, поэтому строки могут быть разной высоты. Колонки таскаются мышью и закрепляются слева/справа. Не хочется собирать разметку руками — есть готовый `<Windowgrid>`.

Демка крутит миллион строк: `cd demo && pnpm dev`. Тесты — `pnpm test`.
